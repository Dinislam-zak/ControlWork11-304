Вариант 1.

Разработать структуру программы управления лифтами, протестировать ее.
Задачи:
1. Структура для определения свободного лифта (интерфейс с методом Elevator free(),
класс реализующий этот интерфейс через закольцованный связный список  )
2. Структуры, описывающие Лифт (Абстрактный класс лифт (Elevator), лифт для четных этажей, лифт для нечетных этажей, лифт для сотрудников), статус (перечисление) — занят, свободен
3. Интерфейс Вызов, описывающий вызов лифта, в зависимости от ситуации (Перечисление: нужен четный этаж, нечетный, сотрудник)
4. Генерация и обработка исключения, если все лифты заняты

Тест:
Создание 3 лифтов для четных этажей
Создание 3 лифтов для нечетных этажей
Создание 1 лифта для сотрудников

Оформление 20 вызовов лифта
